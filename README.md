# 歡迎加入 TubeHR — 你的第一天

你今天加入 **TubeHR** 的 Foundation Team，負責核心人事模組。

這是你的 onboarding 任務包。裡面有：

| 資料夾 | 內容 |
|--------|------|
| `codebase/` | 模組的核心程式碼（6 個檔案） |
| `team-chat/` | 團隊 Slack 頻道的近期訊息 |
| `backlog/` | 目前的待辦事項（3 張卡片） |
| `your-work/` | 你的工作區（交付放這裡） |

---

## 你的任務

**用 3 小時，做你認為最重要的事。**

沒有規定順序，沒有必須完成的項目。你自己決定：
- 先讀什麼
- 做什麼、不做什麼
- 怎麼分配時間

我們想看的是你**怎麼思考和工作**，不是你能做多快。

---

## 工具

你可以使用任何工具，包括：
- **AI 助手**（ChatGPT、Claude、Copilot 等）
- **搜尋引擎**
- **任何你習慣的開發工具**

唯一的要求：在 `your-work/WORKLOG.md` 中記錄你的工作過程。
不需要鉅細靡遺，但至少讓我們看到你的**思路、決策、和取捨**。

---

## 交付

把所有產出放在 `your-work/` 資料夾。格式不限，但建議：

- `WORKLOG.md` — 你的工作日誌（**必須**）
- 其他任何你認為該產出的東西

`WORKLOG.md` 最少要回答：
1. 你讀完資料後，最先做的事是什麼？為什麼？
2. 你做了哪些取捨（選擇做 A 而不是 B）？
3. 哪些地方你用了 AI？AI 有沒有幫倒忙？
4. 如果再給你 3 小時，你接下來會做什麼？

---

## 面試時

我們會根據你的交付進行 30~45 分鐘的對談。重點不在「答對」，而在：
- 你怎麼決定優先順序
- 你怎麼在不確定的情況下推進
- 你怎麼跟團隊協作（透過你留的 PR comment、提問、文件看出來）
- 你怎麼善用 AI 又不被 AI 帶偏

祝你順利。有問題隨時在 `your-work/QUESTIONS.md` 中記錄。
